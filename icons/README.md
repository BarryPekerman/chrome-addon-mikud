# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Icon Requirements

- Format: PNG
- Background: Transparent or solid color
- Design: Should represent zip code/postal code search functionality
- Text: Can include "מיקוד" or "Mikud" text if desired

## Creating Icons

You can create icons using:
- Image editing software (GIMP, Photoshop, etc.)
- Online icon generators
- Design tools like Figma or Canva

For now, you can use placeholder images or create simple colored squares with text as temporary icons until final designs are ready.
